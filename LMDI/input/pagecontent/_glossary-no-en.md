# Terminologi: norsk–engelsk

Arbeidsdokument for konsistent oversettelse av LMDI IG. Forankres med kollega før bulk-oversettelse av FSH-filer.

| Norsk | Engelsk |
|---|---|
| Legemiddeldata fra institusjon til Legemiddelregisteret | Medication data from institutions to Legemiddelregisteret |
| Legemiddelregisteret (LMR) | Legemiddelregisteret (LMR) |
| Folkehelseinstituttet (FHI) | Norwegian Institute of Public Health (NIPH) |
| Legemiddel | Medication |
| Legemiddelrekvirering | Medication request |
| Legemiddeladministrering | Medication administration |
| Virkestoff | Substance |
| Pasient | Patient |
| Helsepersonell | Practitioner |
| Organisasjon | Organization |
| Diagnose | Condition |
| Kontakt / Behandlingsepisode | Encounter |
| Fødselsnummer | National identity number |
| D-nummer | D-number |
| Klinisk studie | Clinical trial |
| Behandlingsregime | Treatment regimen |
| Prosentvis doseendring | Percentage dose adjustment |
| Cellegift / Kjemoterapi | Chemotherapy |
| SignertKryptertBundle | Signed and Encrypted Bundle |
| Selvadministrering | Self-administration |
| Innleggelse / Poliklinikk | Inpatient / Outpatient |
| Sykehjem | Nursing home |

**Egennavn som beholdes uendret:** Legemiddelregisteret, Maskinporten, HelseID, no-basis, FEST, FHI, LMR, LMDI, HPR, RESH

**Skal ikke oversettes:** canonical-URLer, profil-IDer, instance-IDer, `code`/`system`-verdier, OID-er, FHIR-elementnavn
